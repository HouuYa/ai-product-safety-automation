> AI를 활용한 정부 리콜제품 정보 관리 엔드투엔드 자동화 시스템.
> OECD 글로벌 리콜 포털 연동, n8n 워크플로우, Supabase 데이터 파이프라인, 멀티모달 AI 분석, 실시간 온라인 유통 감시를 통합합니다.

> **AI 자동화로 정부 업무 혁신: 비용 95% 절감, 처리시간 83% 단축**

---

## 🎯 개요

AI를 활용한 엔드투엔드 자동화 시스템:

- **43개국** 리콜 제품 정보 수집·관리
- **OECD 글로벌 리콜 포털**과 실시간 연계
- **온라인 쇼핑몰** 리콜제품 유통 및 사고징후 자동 감시
- **뉴스 포털**에서 제품사고 실시간 탐지
- **소비자 후기·문의** 분석을 통한 조기 경보

**전문 개발자가 아닌 공무원 실무자가 직접 구축** — 현대 AI와 Low-Code 도구를 활용하면 비기술자도 엔터프라이즈급 자동화 시스템을 구축할 수 있음을 증명했습니다.

## 🚀 주요 기능

### 1. 국내 리콜 → OECD 자동 등록

국내 리콜 정보를 OECD 글로벌 리콜 포털에 자동 등록하는 파이프라인.

**프로세스:**
1. 제품안전정보 포털(`safetykorea.kr`) 매일 자동 모니터링
2. API를 통한 신규 리콜 정보 추출
3. GPT-4로 영문 번역 + 이미지 리사이징
4. GPC(국제 표준 제품분류) 코드 AI 자동 부여
5. OECD 요구 XML 형식 자동 생성
6. 담당자 최종 검토(5분) → `globalrecalls.oecd.org` 자동 제출

**성과:** 8년간 외주 계약 종료, 연간 1,900만원 절감

[자세히 보기 →](./docs/01-domestic-recall-oecd.md)

---

### 2. 해외 리콜 정보 수집 (43개국, 시범 구축)

다국어 스크래핑 및 AI 분석을 통한 국제 리콜 데이터 수집.

**수집 국가:** 미국(CPSC), 캐나다, 일본(경제산업성·리콜플러스), 중국(SAMR), 호주(ACCC), 뉴질랜드, EU 세이프티게이트(27개국)

**프로세스:**
1. 13개 정부 웹사이트 매일 스크래핑 (리콜 공표문·이미지·PDF)
2. 다국어 번역 (영어·중국어·일본어·독일어·프랑스어)
3. 위해 유형·화학물질 AI 분류
4. Supabase 구조화 DB 저장
5. 중대 리콜 → Telegram 실시간 알림

**성과:** 처리 시간 4일 → 1시간 이내 (75% 단축)

[자세히 보기 →](./docs/02-overseas-recall-collection.md)

---

### 3. 온라인 쇼핑몰 유통 감시

AI 기반 리콜제품 판매 탐지 및 사고 징후 분석.

**프로세스:**
1. ScrapingBee + 쿠키 기반으로 쿠팡·네이버쇼핑 상품정보·이미지·후기 자동 수집
2. 리콜 DB와 교차 검증 (제조사·모델명·KC 인증번호)
3. GPT-4 Vision 멀티모달 분석
   - 제품 이미지 vs 리콜 공표 이미지 비교
   - 소비자 후기에서 사고 징후 키워드 탐지
4. 위해성 평가 보고서 생성 → 담당자 최종 판단

**성과:** 분석 시간 60% 단축, 누적 리콜제품 재유통 상시 감시 가능

[자세히 보기 →](./docs/03-marketplace-surveillance.md)

---

### 4. 제품사고 뉴스 모니터링

제품사고 뉴스 자동 탐지 및 즉시 전파.

**프로세스:**
1. 네이버 뉴스·소방방재신문 **2시간 주기** 자동 모니터링
2. AI 기반 사고 분류 (화재·폭발·감전·중독 등)
3. 제품명·위해 내용·출처 자동 추출
4. Telegram으로 관계자에게 즉시 전파

**성과:** 주요 제품사고 정보 실시간 공유 체계 구축, 인명피해 등 중대 사고 즉시 대응 가능

[자세히 보기 →](./docs/04-news-monitoring.md)

## 🛠 기술 스택

| 계층 | 기술 | 용도 |
|------|------|------|
| **자동화** | n8n Cloud | 워크플로우 오케스트레이션 |
| **AI/ML** | ChatGPT, GPT-4 Vision, Gemini | 번역·분류·이미지 분석 |
| **데이터베이스** | Supabase (PostgreSQL) | 구조화 데이터 저장·REST API 자동 생성 |
| **스크래핑** | ScrapingBee, Puppeteer | 봇 방지 우회·쿠키 기반 데이터 추출 |
| **API** | REST, RSS, Webhooks | 시스템 통합 |
| **알림** | Telegram API | 실시간 담당자 알림 |

> **Gemini 선택 이유:** 무료 티어의 높은 요청 한도 → 예산 제한 없이 빠르게 PoC 가능

- ✅ 선제적 유통 감시 체계 (누적 리콜제품 재유통 차단)

## 🎓 조직 AI 전환(AX) 역량 내재화

시스템 구축을 넘어 조직 내 AI 역량을 배양했습니다:

- 6회 실습 워크샵 개최 (100명+ 교육)
- Low-Code/No-Code 도구 도입
- AI 참고 도서 27권 배포

**핵심 통찰:** 공무원도 적절한 도구와 교육만 주어지면 AI 솔루션을 직접 구축할 수 있다 — "공무원은 코딩 못한다"는 편견을 깨는 것이 이 프로젝트의 또 다른 성과입니다.
